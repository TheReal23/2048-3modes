2048 - 3 Modes

CRAZY MODE: This mode is the same as the original 2048, but it will add a new tile every 0.5 second. Survival as long as possible. You can play it via thereal23.github.io/2048-3modes/crazy.html.

NASTY ZER0 MODE: There are some zero tile added in this mode. The zero tile can't be remove, but it can join other zero tile and get a high score more than other numbers. Clear zero and get more score! You can play it via thereal23.github.io/2048-3modes/nasty0.html.

TIME TRIAL MODE: In this mode, the game time is limited to 360 seconds. You have to get to 2048 in less than 360 seconds. You can play it via thereal23.github.io/2048-3modes/trial.html.

Or you can go back to the original 2048 via thereal23.github.io/2048-3modes.
